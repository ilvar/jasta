jasta

Jabber status bot

  1. Install in virtualenv, pip install -r pip_requirements.txt
  2. Install redis
  3. Copy bot/local_config.py.dist to bot/local_config.py and set up an account for bot
  4. Run python runbot.py
  5. Add bot to your roster
  6. Run python runserver.py
  7. Navigate http://127.0.0.1:500/your_jid@server.com/ in your browser to see your current status
